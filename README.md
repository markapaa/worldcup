# World Cup 2026 Analytics Platform

Ένα διαδραστικό web app ανάλυσης δεδομένων και προβλέψεων για το Παγκόσμιο Κύπελλο του 2026, χτισμένο με Streamlit και Scikit-learn.


## Τι κάνει το App

1. **Match Predictor:** Προβλέπει τον νικητή ανάμεσα σε 48 ομάδες. Χρησιμοποιεί **Logistic Regression** και λαμβάνει υπ' όψιν του τη δύναμη επίθεσης/άμυνας, το πλεονέκτημα έδρας (για ΗΠΑ, Μεξικό, Καναδά) και προσαρμόζεται μαθηματικά στα ουδέτερα γήπεδα.
2. **Player Scouting:** Βρίσκει τον μαθηματικό "σωσία" κάθε παίκτη της διοργάνωσης. Χρησιμοποιεί **K-Nearest Neighbors (KNN)** για να συγκρίνει στατιστικά (αξία, γκολ, λεπτά κ.λπ.) και εμφανίζει ένα διαδραστικό **Radar Chart** (Plotly) με την ομοιότητά τους.

## Τεχνολογίες
`Python`, `Pandas`, `Scikit-learn`, `Plotly`, `Streamlit`
